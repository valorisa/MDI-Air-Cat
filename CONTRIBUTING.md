# Contribuer au projet MDI CAT

Merci de t'intéresser à ce projet pédagogique.

## Ce qui est bienvenu

- 🐛 **Corrections de bugs** : si un calcul ne correspond pas au PDF ou si une équation est mal transcrite
- 📚 **Améliorations pédagogiques** : reformulation d'un passage, ajout d'une analogie, correction d'une faute de frappe
- 🔬 **Extensions** : simulation dynamique pas-à-pas, benchmark avec batteries modernes, analyse historique

## Ce qui n'est pas bienvenu

- ❌ **Curve fitting** : toute PR qui modifie les constantes du PDF (rendement 85%, pertes 2%) pour "faire coller" les chiffres sera rejetée
- ❌ **Suppression des avertissements** : les notes sur les limites du modèle (gaz parfait, fallback CoolProp) sont essentielles

## Comment contribuer

1. Fork le dépôt
2. Crée une branche : `git checkout -b feature/ma-contribution`
3. Teste ton notebook en local ou sur Colab
4. Ouvre une Pull Request avec une description claire de ce que tu as changé et pourquoi
